# Security Policy

## Supported Scope

当前仓库主要面向：

- macOS 13+
- Apple Silicon 优先
- 本地语音识别与本地文本注入

## 报告方式

如果你发现了安全问题，请不要直接公开提交带有利用细节的 issue。

建议通过私下方式联系维护者，并至少包含以下信息：

- 问题描述
- 影响范围
- 复现步骤
- 可能的修复建议

## 常见高风险区域

以下能力天然涉及更高的权限边界：

- 麦克风访问
- 辅助功能权限
- 输入监控权限
- 跨应用文本注入

如果你提交和这些区域相关的改动，请尽量说明：

- 是否引入新的系统权限依赖
- 是否扩大了跨应用操作范围
- 是否会把本地数据发送到外部服务

## 当前安全边界

当前默认设计目标是：

- 语音识别在本地完成
- 不依赖云端转写
- 不主动上传录音内容

但首次运行仍然需要用户手动授予系统权限。
